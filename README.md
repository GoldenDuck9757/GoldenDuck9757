### 2. **Bot de Discord com Memória Avançada**
- **Descrição**: Criei um bot de Discord capaz de manter o contexto de conversas por longos períodos, respondendo de forma inteligente e natural. Ele utiliza o Google Generative AI para garantir respostas precisas e adaptadas ao contexto.
